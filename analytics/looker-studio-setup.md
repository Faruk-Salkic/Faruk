# Looker Studio Report Setup Guide

## Data Sources to Connect
- Google Analytics 4 (native connector)
- Google Search Console (native connector)
- Google Ads (native connector)
- Manual data via Google Sheets if needed

## Recommended Report Structure

### Page 1: Executive Summary
- Date range control at top
- Scorecards: Sessions, Conversions, Conv. Rate, Cost Per Lead
- Line chart: Sessions trend 90 days
- Bar chart: Top 5 channels by conversions

### Page 2: Traffic
- Sessions by channel (table + pie chart)
- Session trend by channel (line)
- Top landing pages by sessions
- New vs returning split

### Page 3: Conversions
- Conversion funnel
- Conversions by source/medium
- Conversion rate by device

### Page 4: SEO
- Clicks and impressions trend
- Top 20 queries by clicks
- Top pages by impressions
- Average position by page

## Sharing with Clients
- Add client email as Viewer
- Schedule email delivery: Monday 8am
- Add text boxes explaining key charts for non-technical clients

## Pro Tips
- Use Blend Data to combine GA4 + Ads spend
- Date comparison controls let clients self-serve
- Match theme colours to client brand
