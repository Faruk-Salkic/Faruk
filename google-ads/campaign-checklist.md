# Google Ads Notes

> Working notes for campaigns @ alooweb.de

## Campaign Setup Checklist
- [ ] Keyword list finalised (50+ exact/phrase)
- [ ] Negative keywords imported
- [ ] Sitelinks, callout, snippet extensions set
- [ ] Conversion tracking verified in GA4
- [ ] Budget pacing set to Standard
- [ ] Audience segments added (observation mode)

## Ad Copy Rules
- H1: Primary keyword
- H2: USP / offer
- H3: CTA or brand name
- D1: Benefits + keyword
- D2: Social proof / urgency
- Always pin H1 and D1

## Weekly Optimisation
- Search term report → add negatives
- Pause keywords with CTR < 1% after 200 impressions
- A/B test one ad variant per week
- Check impression share lost to budget vs rank

## Target Metrics
| Metric | Goal |
|--------|------|
| CTR | > 5% |
| Quality Score | >= 7 |
| Conv. Rate | > 3% |
| ROAS | > 4x |
