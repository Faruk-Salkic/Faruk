# Negative Keywords Master List

## Universal Negatives (Add to Every Campaign)
- free, cheap, discount, coupon, diy, tutorial, how to
- jobs, career, salary, hiring, intern
- wikipedia, reddit, forum
- download, crack, torrent, pdf

## Research Intent Negatives
- what is, definition, meaning, example
- template, sample, format

## How to Build Your List
1. Run campaign for 2 weeks
2. Open Search Terms report
3. Filter: zero conversions + spend above threshold
4. Add irrelevant terms as negatives
5. Repeat every week

## Match Type Guide
- Broad negative: blocks many variants — good for obvious irrelevant terms
- Phrase negative: use for multi-word negatives
- Exact negative: surgical — use when most traffic is good but one variant is not
