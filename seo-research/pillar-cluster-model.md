# Pillar-Cluster Content Model

## Concept
One broad PILLAR page covers a topic at high level.
Multiple CLUSTER pages cover sub-topics in depth.
All clusters link to the pillar. Pillar links to all clusters.

## Example
Pillar: Google Ads Guide (keyword: google ads)
Clusters:
- Google Ads bidding strategies
- Google Ads quality score
- Google Ads negative keywords
- Google Ads for ecommerce

## Building the Model
1. Pick 5-10 topics your site should own
2. Find 8-15 sub-topic keywords per topic
3. Map existing content — already covered?
4. Create briefs for each content gap
5. Interlink all pieces once published

## Internal Linking Rules
- Anchor text = target keyword of destination page
- Link from high-authority pages to new content
- Every cluster must link back to its pillar
- Add links in body copy, not just sidebars
