# Keyword Research Notes

## Intent Types
| Tier | Intent | Example |
|------|--------|--------|
| 1 | Transactional | buy X, X price, hire X |
| 2 | Commercial | best X, X vs Y, X review |
| 3 | Informational | how to X, what is X |
| 4 | Navigational | brand name |

## Research Process
1. Seed keywords from client brief
2. Expand via Google Suggest + PAA
3. Filter by intent
4. Cluster by topic (pillar + cluster)
5. Map to existing pages or flag for new content

## Content Gap Checklist
- [ ] Compare top 3 competitor pages
- [ ] Check featured snippet opportunities
- [ ] Identify PAA questions we can answer
- [ ] Find keywords competitors rank for that we do not

## Tools
- Google Search Console — position tracking
- Ahrefs / Ubersuggest — volume and difficulty
- AnswerThePublic — question queries
- Google Trends — seasonal patterns
