# Technical SEO Checklist

## Crawling and Indexing
- [ ] robots.txt allows key pages
- [ ] XML sitemap submitted to Search Console
- [ ] No important pages returning 4xx or 5xx errors
- [ ] Canonical tags correct
- [ ] Hreflang set up if multilingual

## Core Web Vitals
- [ ] LCP under 2.5s
- [ ] INP under 200ms
- [ ] CLS under 0.1
Tools: PageSpeed Insights, GTmetrix, CrUX in GSC

## On-Page Basics
- [ ] Unique title tag per page (50-60 chars)
- [ ] Unique meta description per page (150-160 chars)
- [ ] H1 present and includes keyword
- [ ] Images have descriptive alt text
- [ ] No duplicate content

## Mobile
- [ ] Mobile-friendly test passes
- [ ] Tap targets not too small
- [ ] No horizontal scrolling
- [ ] Font size at least 16px body text

## Schema Markup
- Organization schema on homepage
- Article schema on blog posts
- Product schema on product pages
- FAQ schema where applicable
