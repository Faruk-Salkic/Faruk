# Social Media Notes

## Weekly Content Mix
| Day | Format | Theme |
|-----|--------|-------|
| Mon | Carousel | Educational |
| Tue | Reel | Behind the scenes |
| Wed | Text post | Opinion |
| Thu | Infographic | Data / stats |
| Fri | Story poll | Engagement |

## Caption Structure
1. Hook — must show before read more
2. Value / story (2-4 short paragraphs)
3. CTA — question or link in bio
4. Hashtags (10-15, mix niche + broad)

## Hook Formulas That Work
- Most people overlook this...
- We tested X for 30 days. Results:
- Stop doing [X]. Do this instead:
- [Number] things I wish I knew about [topic]

## Platform Notes
- Instagram: Reels 9:16, 5-7 posts/week
- LinkedIn: Long text + document carousels
- TikTok: Hook must land in first 2 seconds
- Facebook: Video + events still work for local
