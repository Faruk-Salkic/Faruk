# Instagram Reels Checklist

## Pre-Production
- [ ] Hook concept written — first 2 seconds must stop scroll
- [ ] Script max 60 seconds read aloud
- [ ] Trending audio identified (check Reels tab weekly)
- [ ] Filming location and lighting sorted

## Hook Formats That Work
- Bold text claim on black screen
- Jump cut mid-action, skip the intro
- Question format: Why does nobody talk about X?
- Surprising stat displayed on screen

## Filming
- [ ] Vertical 9:16 format
- [ ] 4K if possible
- [ ] Use tripod or stabiliser
- [ ] Film 3x more than needed, cut later

## Editing
- [ ] Captions auto-generated and proofread
- [ ] Silences and filler words cut
- [ ] First frame works as thumbnail
- [ ] Text in safe zones (avoid bottom 20%)

## Publishing
- [ ] Posted between 7-9am or 6-9pm local time
- [ ] Caption hook visible before read more
- [ ] 10-15 hashtags added
- [ ] Shared to Stories within 1 hour
- [ ] Replied to early comments within first hour
