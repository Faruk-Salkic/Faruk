# Prompt Library — Marketing

## Ad Headlines
```
Act as a Google Ads copywriter.
Write 10 headlines (max 30 chars each) for [product/service] targeting [audience].
Main keyword: [keyword]
Focus: USP, benefit, urgency, social proof.
Format: numbered list only.
```

## Email Subject Lines
```
Write 15 email subject lines for a campaign about [topic] targeting [audience].
Mix styles: curiosity, urgency, personal, list, negative frame.
Max 50 characters each.
Mark top 3 with an asterisk.
```

## Social Caption
```
Write an Instagram caption for a post about [topic].
Brand voice: [describe]
First line must be a hook that works before read more.
End with a question to drive comments.
Add 10 relevant hashtags separately.
```

## Competitor Page Analysis
```
Analyse this page: [paste text or URL]
1. Main value proposition
2. Objections they address
3. Trust and proof elements used
4. CTA structure
5. What you would test or improve
```

## Content Calendar
```
Create a 4-week social media calendar for [brand/niche].
Platform: [platform]
Frequency: [X per week]
Brand voice: [describe]
Include: topic, format, hook idea per post.
Format as a table.
```
